# Healthcare Data Analytics Project

## Overview
This project is a completed university assignment focused on analyzing hospital patient data with Python and machine learning. It combines data cleaning, exploratory analysis, predictive modeling, and result interpretation to show how analytics can support better healthcare decisions.

## What I Built
- A healthcare analytics workflow using 472 anonymized patient records from a small hospital.
- Data preprocessing steps including missing value handling, categorical encoding, scaling, and grouping rare disease classes.
- Exploratory analysis of age, gender, symptoms, and service-related factors such as waiting time and administration.
- Machine learning models for satisfaction classification, disease prediction, patient clustering, and hospital stay regression.
- Evaluation outputs using classification metrics, confusion matrices, and regression error measures.

## Skills Demonstrated
- Data cleaning and preprocessing.
- Exploratory data analysis.
- Feature encoding and scaling.
- Classification, clustering, and regression modeling.
- Model evaluation and interpretation.
- Translating analytical results into healthcare-focused recommendations.

## What I Learned
This assignment strengthened practical skills in building an end-to-end analytics workflow, from preparing messy data to evaluating machine learning models. It also reinforced how important data quality, balanced classes, and realistic target variables are when working on healthcare analytics problems.

## Limitations
- This is a coursework assignment and is not being actively developed.
- The dataset is relatively small and includes imbalanced classes, which limited model performance.
- The regression task used a synthetic target for hospital stay, so those results are not suitable for real-world prediction.
